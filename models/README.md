# Modelos

Aqui va el modelo LSTM entrenado y sus artefactos relacionados.

Ejemplos:

- `lstm_model.pth`
- `scaler.pkl`
- `columns.json`
- `metadata.json`

Si el modelo pesa mas de 100 MB, GitHub no lo acepta como archivo normal. En ese caso usa Git LFS o subelo como release y descargalo desde Render/GitHub Actions.
